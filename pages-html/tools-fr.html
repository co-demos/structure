Nous développons différents logiciels open source

Ces outils permettent d’extraire, de consolider et de visualiser les informations.

Nous investissons dans de l’innovation numérique. Cette suite est open-source et conçue de manière générique. Elle peut être adaptée à d’autres projets. Les différents logiciels fonctionnent indépendamment.


  • Open scraper collecte toute l’information des projets sur les sites des membres du collectif.

  • Solidata centralise et consolide les informations sur les projets.

  • Apiviz est l’interface qui permet de visualiser ces informations sous forme de fiches projets, cartographie et datavisualisation.

  • TokTok est un microservice d'authentification.



Open Scraper

Pour collecter des données publiées sur des sites

Fonctionnalités

  • Récupérer des données publiées sur des sites internet
  • Homogénéiser des données disparates
  • Ouvrir les données en maîtrisant leurs degrés d'ouverture (des données qui restent privées à de l’open data)

Solidata

Pour faciliter l'ouverture de données

Fonctionnalités

  • Agréger des données provenant de sources ou de fichiers différents
  • Enrichir les données grâce à des données externes : géolocalisation, données administratives (SIRET), ...
  • Homogénéiser les tags des données agrégées
  • Ouvrir les données en maîtrisant leurs degrés d’ouverture (des données qui restent privées à de l’open data)

Apiviz

Fonctionnalités

  • Visualiser les données agrégées sous différentes formes : cartographie, datavisualisation, vue détaillée des données (fiches projets) …
